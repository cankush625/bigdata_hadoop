mapred_job_tracker
==================

Ansible role for configuring MapReduce Job Tracker.

Requirements
------------

OS: Amazon Linux 2, RHEL8 <br>
CPU: 1 Cores <br>
RAM: 1GiB

Role Variables
--------------

No any variables

Dependencies
------------

No any dependency

Example Playbook
----------------

The example playbook below will show how to use this role:

    - hosts: jobtracker
      roles:
        - role: mapred_job_tracker
